# מדריך התקנה לצוות — תיקון תמלולים עם Claude Code

מדריך זה מסביר איך להגדיר את הפרויקט על המחשב שלך פעם אחת, ואיך לבקש מ-Claude Code
לתקן שיחות/פגישות שלא תומללו.

## חלק א׳ — התקנה חד-פעמית (במחשב Mac)

1. **לקבל גישה לפרויקט** — תקבלי הזמנה במייל מ-GitHub. אשרי אותה.
2. **להתקין כלים** (פעם אחת). פתחי את אפליקציית **Terminal** והדביקי:
   ```bash
   # התקנת Homebrew (אם אין):
   /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
   brew install git python@3.12 ffmpeg
   ```
3. **להוריד את הפרויקט**:
   ```bash
   cd ~/Desktop
   git clone https://github.com/gadtamir/zoom-ghl-notes.git
   cd zoom-ghl-notes/server
   python3.12 -m venv .venv
   ./.venv/bin/pip install -r requirements.txt
   ```
4. **קובץ המפתחות** — בקשי מגד את הקובץ `.env` (הוא מכיל סיסמאות/מפתחות — לא נשלח דרך
   GitHub). שמרי אותו בתוך התיקייה `server/` בשם `.env` בדיוק. תבנית לדוגמה: `server/.env.example`.

זהו. ההתקנה הסתיימה.

## חלק ב׳ — איך לבקש עזרה מ-Claude Code

פתחי **Claude Code בתוך תיקיית הפרויקט** (`zoom-ghl-notes`) ופשוט תארי את הבעיה
במילים שלך. תמיד תני **אימייל או טלפון + תאריך**. דוגמאות להעתקה:

- *"השיחה של [אימייל/טלפון] מתאריך [תאריך] לא קיבלה סיכום ב-GHL. תבדוק למה ותתקן."*
- *"תבדוק אם יש שיחות מהימים האחרונים שלא תומללו, ותתמלל את אלה שיש להן הקלטה."*
- *"תבדוק שהמערכת תקינה — שיש קרדיט ב-OpenAI וב-Anthropic."*

Claude יאבחן (קרדיט? יש הקלטה? קצרה מדי?), יסביר, ויתקן. הוא יודע להריץ את
הסקריפטים לבד — את לא צריכה לדעת פקודות.
